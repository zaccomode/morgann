Targeting users

Suggested Usage:

Morgann target @user

Every time the mentioned user sends a message, Morgann will automaticaly reply with an insult.


Variations:

Morgann target off

Morgann will stop targeting any users, and disable non-admins from targeting anyone until somebody is targeted again.

Admin Feature

Insults

Adding an Insult:

Morgann add insult [your insult]

Adds an insult to the pool of insults used when targeting a user.

This pool is server-specific.


Searching & Removing:

Morgann search insult [insult]

OR

Morgann random insult

Shows an insult found using the search query.
