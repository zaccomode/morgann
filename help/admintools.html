Channel Binding

I understand that Morgann can become irritating and out of control, especially if targeted members of your server are active. For that reason, you can bind Morgann to a specific channel.

Morgann bind channel here

Morgann will no longer insult targeted people, show quotes or respond to commands outside of that channel.

This can be undone by using:

Morgann bind channel off

Remove Target

Morgann target off

Stops targeting anybody, and also disables non-Admins from targeting people until an Admin targets someone again (using the usual command).

Other Admins

Upon joining a server, Morgann will only have his Admin Tools available to the server owner. However, the owner may add any users with a particular role using the following command:

Morgann set [mod|admin] role @role

It goes without saying that this role must be mentionable by you, even if only temporary.

Anybody with that role can now use any tools mentioned on this page.

Any admins may also change the required mod role with the above command, or remove the mod role using the following command, which will relinquish power exclusively to the server owner.

Morgann set [mod|admin] role off
